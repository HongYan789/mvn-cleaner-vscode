# MAVEN CLEANER

A simple tool to clean your local maven repository

# Usage:

1. plugin center download 'maven cleaner' plugin
2. Run the Command Maven Cleaner or right-click Maven Cleaner or use the shortcut Ctrl + Command + C
3. choose maven path
4. save
